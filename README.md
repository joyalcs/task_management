# ✅ Task Management System  

A simple yet powerful **Task Management System** built with **Django REST Framework** 🚀  

---

## ✨ Features
- 🔐 Role-based Access (SuperAdmin, Admin, User)
- ✅ Task Creation, Assignment & Tracking
- 📊 Task Completion Reports
- 🛠 Built with **Django + DRF**

---

## ⚡ Installation Guide

### 1️⃣ Clone the repository
```bash
git clone https://github.com/joyalcs/task_management.git
cd task_management_backend
# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```
